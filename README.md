# 앱 연구소 · App study

초등학생이 앱의 작동 원리를 쉽고 재미있게 탐구할 수 있도록 만든 웹 활동지입니다.  
학생은 자신이 고른 앱을 관찰하고, **화면에 있는 것 → 사용자의 행동 → 앱의 반응 → 사용자의 결과** 흐름으로 정리하며 앱이 어떻게 작동하는지 이해할 수 있습니다.

## 주요 기능

- **활동 1: 앱 사용 이야기 만들기**  
  누가, 어떤 상황에서 앱을 사용하는지 짧은 이야기로 정리합니다.

- **활동 2: 앱 작동 분석하기**  
  화면에 있는 것, 사용자의 행동, 앱의 반응, 사용자의 결과를 차례대로 작성하면 알고리즘 문장이 자동으로 완성됩니다.

- **활동 3: 앱 연구원의 한마디**  
  앱의 좋은 점, 아쉬운 점, 개선 아이디어를 문장으로 정리합니다.

- **자동 저장**  
  작성 내용과 현재 단계가 브라우저 로컬스토리지에 저장되어 새로고침 후에도 이어서 작성할 수 있습니다.

- **백업/복원**  
  우측 상단 저장 아이콘에서 JSON 백업 파일을 내려받거나, 기존 JSON 파일로 작성 내용을 복원할 수 있습니다.

- **결과물 내보내기**  
  완성된 결과물을 텍스트 복사, 이미지 복사, 텍스트 저장(.txt), 이미지 저장(.jpg) 방식으로 공유할 수 있습니다.

## 사용 방법

1. `index.html` 파일을 브라우저에서 엽니다.
2. 분석할 앱 이름과 모둠/이름을 입력합니다.
3. 활동 1~3을 차례대로 작성합니다.
4. 결과 화면에서 필요한 방식으로 결과물을 복사하거나 저장합니다.
5. 장기 보관이 필요하면 우측 상단 저장 아이콘에서 JSON 백업을 저장합니다.

## 아이디어 존중 요청

이 프로젝트는 학생들이 앱과 기술을 더 쉽게 이해하도록 돕기 위한 교육용 아이디어와 활동 흐름을 담고 있습니다.  
수업, 연구, 교육 자료 제작에 활용하실 때는 원래의 기획 의도와 창작 아이디어를 존중해 주세요.

- 원저작자와 출처를 가능한 범위에서 밝혀 주세요.
- 활동 구조나 문구를 바탕으로 2차 자료를 만들 때는 무단 도용처럼 보이지 않도록 충분히 변형하거나 인용을 표시해 주세요.
- 학생들이 더 쉽게 배우도록 개선한 아이디어가 있다면 함께 나누어 주세요.
